# Zerune Studio

가사 없는 콘셉트 플레이리스트를 반복 제작하기 위한 로컬 노드형 기획 도구입니다. 완성 음악이나 이미지를 직접 생성하지 않고, 키워드에서 Mureka 프롬프트·30초 자체 모티프 MIDI·이미지 프롬프트·트랙 구조·참고 프레임·YouTube 입력 패키지를 파생합니다.

## 시작하기

`index.html`을 브라우저에서 열면 됩니다. 참고 이미지 미리보기와 원출처를 볼 때는 인터넷 연결이 필요합니다.

1. 상단에서 `랭면이 불었습니다.` 또는 `BEFORE THE NEXT FALL`을 선택합니다.
2. `1 콘셉트 → 2 키워드 → 3 사운드+구조 → 4 프롬프트 → 5 입력` 순서로 이동합니다.
3. 3단계에서 30초 자체 모티프 MIDI를 내려받고, 프로젝트별 길이와 트랙 수를 정한 뒤 `결과 다시 만들기`를 누릅니다.
4. 4단계에서 추천 메인 Mureka 프롬프트를 복사하고 필요한 트랙을 펼쳐 한 번에 한 막씩 트랙별 구조 프롬프트를 복사합니다.
5. 음악과 연결된 추천 이미지 프롬프트 및 렌즈·노출·시간대 변주를 확인합니다.
6. 키워드로 찾은 참고 프레임 10개에서 구도 재설계 아이디어를 고릅니다.
7. 5단계에서 영상 제목·통합 설명·수동 챕터·재생목록·태그·고정 댓글·썸네일 사양·Studio 설정을 입력칸별로 복사합니다.
8. `전체보기`에서 1–5단계의 핵심 결과를 한 번에 확인하거나 전체 프로젝트 텍스트를 복사합니다.
9. 프로젝트를 `JSON`으로 백업하거나 전체 제작안을 `Markdown`으로 내보냅니다.

한 화면에는 현재 단계의 노드만 나타납니다. 노드는 움직이거나 `◇` 버튼으로 잠글 수 있고, 위치와 값은 단계·프리셋별로 브라우저 `localStorage`에 자동 저장됩니다.

## 기본 전략값

- `지배인 동지, 랭면이 불었습니다.`: 30분·5막
- `BEFORE THE NEXT FALL`: 60분·10곡, 공식 10부작과 구조적으로만 대응하며 에피소드 내용은 예측하지 않음
- 30분 류경호텔 구조: `0:00 → 3:54 → 11:24 → 18:00 → 25:12 → 30:00`
- 트랙 수를 바꾸면 구조와 Mureka 트랙 프롬프트 수가 동시에 바뀜
- 트랙 수를 바꿔도 시작부터 기다림·목소리·절정·마지막까지 전체 서사를 가중치에 맞춰 재배치
- 전 트랙에 같은 추천 메인 프롬프트를 앵커로 붙여 음색·믹스·반복 모티프 일관성 유지
- 6–12초 크로스페이드

평양 여름 프리셋은 제목을 `지배인 동지, 랭면이 불었습니다.` 한 문장으로 고정합니다. 마침표를 포함하며 부제·장르·플레이리스트 문구와 썸네일 문구를 붙이지 않습니다. 음악은 앰비언트·전자음향 70%, 개량 민족악기에서 착안한 절제된 질감 30%를 시작값으로 사용합니다.

두 번째 프리셋은 `BEFORE THE NEXT FALL — Cyberpunk: Edgerunners 2 Fan Playlist`를 추천 제목으로 사용합니다. 썸네일 카피, 10곡의 영문·한글 제목, 96–106 BPM의 사운드 범위, 4음 생존 모티프, 비인가 방송 영상 문구를 함께 제공합니다. 공식 캐릭터·스틸·OST는 생성 재료로 사용하지 않으며 내보내기에 비공식 팬 작품 고지가 포함됩니다.

공개 앰비언트 표본에는 30분대·1시간대·2–3시간대가 모두 있고, 사이버펑크·다크 앰비언트에는 약 55–65분 형식이 반복됩니다. 첫 프리셋은 제작 학습용 30분, 두 번째는 사용자가 정한 55–65분 범위의 중앙값인 60분으로 서로 다른 가설을 검증합니다.

## YouTube 입력 패키지

- 영상 제목은 100자, 설명은 5,000자 제한을 자동 점검합니다.
- 통합 설명에는 작품 소개, `00:00`부터 시작하는 수동 챕터, 작품·팬 고지와 해시태그를 한 번에 넣습니다.
- 태그는 표기 변형과 보조 검색어로만 사용하고 제목·썸네일·설명을 우선합니다.
- Mureka 음악이나 생성형 이미지를 사용한 프로젝트는 `변경되거나 합성된 콘텐츠: 예`를 기본 설정값으로 안내합니다.
- 업로드는 비공개 저장 → 처리·저작권·모바일 표시 확인 → 예약 또는 공개 → 고정 댓글 순서로 진행합니다.
- Studio 계정에 자동 로그인하거나 업로드하지 않으며, 실제 입력 직전 복사하기 좋은 텍스트만 로컬에서 만듭니다.

## Mureka 사용 시 주의

- 이 정적 앱에 API 키를 입력하지 마세요.
- 모든 Zerune 작품은 무가사입니다. 엔진이 모든 메인·트랙 프롬프트의 처음과 끝에 강제 무가사 잠금을 고정해 가사·노래·대사·허밍·코러스·랩·보컬 찹·의미 없는 음절을 금지합니다.
- 생성 프롬프트는 반드시 Mureka의 `Instrumental` 모드에 복사합니다. Song·Lyrics·보컬 생성 모드는 사용하지 않고, 전체 플레이리스트를 한 번에 요청하지 않으며 한 곡씩 생성합니다.
- 처음에는 텍스트 프롬프트만으로 3개를 비교합니다. 음색은 좋은데 반복 모티프가 흔들릴 때만 3단계의 자체 MIDI를 `Melody ideas`에 넣습니다.
- [Mureka 공식 파일 업로드 문서](https://platform.mureka.ai/docs/api/operations/post-v1-files-upload.html)는 `melody` 입력에 5–60초 MP3/M4A/MID를 지원하며 MIDI를 권장하고, `reference` 및 `instrumental` 오디오는 정확히 30초 MP3/M4A를 요구합니다.
- Reference 오디오가 필요하면 자체 MIDI를 렌더링한 뒤 직접 녹음하거나 합성한 공간음만 더합니다. 상업음원, 기존 북한 노래, 애니메이션 OST, 대사, 타인의 연주는 업로드하지 않습니다.
- 트랙 목표 길이가 한 번의 출력보다 길면 Song Extension으로 이어 붙이고, 6–12초 크로스페이드용 앞뒤 여백을 남깁니다.
- [2026-05-13 Mureka 약관](https://www.mureka.ai/static/terms-20260513.pdf)상 유료 크레딧/구독 출력과 무료 출력의 권리가 다르고, 업로드 입력에 필요한 권리를 사용자가 보유해야 합니다. 상업 게시 직전에 최신 약관을 다시 확인합니다.

## 참고 이미지

각 프리셋에 Wikimedia Commons 원출처가 연결된 미리보기 10개가 있습니다. 두 번째 프리셋에는 CCTV 관제벽, CRT, 필름 릴, 영화 카메라, 젖은 도시를 포함합니다. 이 카드는 장소·사물·빛·건축의 관찰 자료이며, AI로 변형했다는 사실만으로 원본 권리가 자동으로 사라지는 것은 아니므로 공개본은 구도·시점·인물·배경·광원을 충분히 새로 설계해야 합니다.
