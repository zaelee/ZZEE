# Zerune Studio

가사 없는 콘셉트 플레이리스트를 반복 제작하기 위한 로컬 노드형 기획 도구입니다. 음악이나 이미지를 직접 생성하는 도구가 아니라, 키워드에서 Mureka 프롬프트·이미지 프롬프트·트랙 구조·참고 프레임을 파생합니다.

## 시작하기

`index.html`을 브라우저에서 열면 됩니다. 참고 이미지 미리보기와 원출처를 볼 때는 인터넷 연결이 필요합니다.

1. 상단에서 `류경호텔 105층` 또는 `도시가 다시 타기 전`을 선택합니다.
2. `1 콘셉트 → 2 키워드 → 3 사운드+구조 → 4 프롬프트` 순서로 이동합니다.
3. 3단계에서 대표편 길이와 트랙 수를 정한 뒤 `결과 다시 만들기`를 누릅니다.
4. 4단계에서 추천 메인 Mureka 프롬프트를 복사하고 필요한 트랙을 펼쳐 트랙별 구조 프롬프트를 복사합니다.
5. 음악과 연결된 추천 이미지 프롬프트 및 렌즈·노출·시간대 변주를 확인합니다.
6. 키워드로 찾은 참고 프레임 10개에서 구도 재설계 아이디어를 고릅니다.
7. 프로젝트를 `JSON`으로 백업하거나 전체 제작안을 `Markdown`으로 내보냅니다.

한 화면에는 현재 단계의 노드만 나타납니다. 노드는 움직이거나 `◇` 버튼으로 잠글 수 있고, 위치와 값은 단계·프리셋별로 브라우저 `localStorage`에 자동 저장됩니다.

## 기본 전략값

- 64분 대표편: 류경호텔 10개, 사이버펑크 11개 기본 트랙
- 트랙 수를 바꾸면 구조와 Mureka 트랙 프롬프트 수가 동시에 바뀜
- 트랙 수를 줄여도 문턱부터 절정·귀환까지 전체 서사를 균등하게 재배치
- 전 트랙에 같은 추천 메인 프롬프트를 앵커로 붙여 음색·믹스·반복 모티프 일관성 유지
- 6–12초 크로스페이드

이는 YouTube의 공식 길이 규칙이 아니라 공개 표본과 배경 청취 맥락을 바탕으로 한 시작 가설입니다. 실제 채널에서는 비슷한 길이의 최근 영상끼리 노출 클릭률, 첫 30초 유지, 평균 시청시간, 핵심 구간 유지율을 비교하세요.

## Mureka 사용 시 주의

- 이 정적 앱에 API 키를 입력하지 마세요.
- 생성 프롬프트를 Mureka의 Instrumental 모드에 복사합니다.
- 먼저 추천 메인 프롬프트를 고정한 다음 각 트랙의 시간·에너지·전경 악기만 다르게 생성합니다.
- 트랙 목표 길이가 한 번의 출력보다 길면 Mureka의 Song Extension을 사용하되 메인 프롬프트와 반복 모티프를 유지합니다.
- 상업 게시 전에는 사용 시점의 Mureka 약관과 출력물 라이선스를 다시 확인합니다.

## 참고 이미지

각 프리셋에 Wikimedia Commons 원출처가 연결된 미리보기 10개가 있습니다. 이 카드는 장소·음식·빛·건축의 관찰 자료이며, 카드마다 새로 적용할 렌즈·노출·시간대 메모가 붙어 있습니다. AI로 변형했다는 사실만으로 원본 권리가 자동으로 사라지는 것은 아니므로, 공개본은 구도·시점·인물·배경·광원을 충분히 새로 설계해야 합니다.
