# 존재하지 않는 공간 — 노드형 제작 스튜디오 명세

## 1. 목적

반복 제작할 수 있는 가사 없는 콘셉트 플레이리스트를 위해, 키워드에서 음악·이미지·구성·배포 계획을 파생하는 로컬 우선(node-based) 기획 도구를 만든다.

첫 프리셋은 다음 두 가지다.

1. **류경호텔 105층, 평양랭면과 과일주** — 차갑고 통제된 연회, 미완성 공간, 금지된 환대.
2. **도시가 다시 타기 전** — 완전히 독창적인 사이버네틱 디스토피아 스틸과 기악곡. `Cyberpunk: Edgerunners 2`의 공식 작품·인물·로고로 오인되지 않게 한다.

이 도구는 음악 또는 이미지를 직접 생성하거나 Mureka API 키를 보관하지 않는다. 사용자가 Mureka와 이미지 생성기에 붙여 넣을 수 있는 프롬프트 및 제작 계획을 만든다.

## 2. 사용자 흐름

1. 프리셋을 고른다.
2. 상단 FLOW에서 `1 콘셉트 → 2 키워드 → 3 사운드+구조 → 4 프롬프트` 순서로 이동한다.
3. 현재 단계에 필요한 노드만 보고 핵심 변수와 키워드를 수정한다.
4. 3단계에서 트랙 수와 구조를 확정하면 4단계에 동일한 수의 트랙별 음악 프롬프트가 생긴다.
5. 메인 음악 프롬프트를 모든 트랙의 일관성 앵커로 쓰고, 각 트랙 프롬프트에는 해당 구간의 구조만 덧붙인다.
6. 이미지 프롬프트는 동일한 키워드·정서·색을 공유하며 음악 프롬프트와 연결해 표시한다.
7. 키워드에서 찾은 참고 이미지 10개를 원출처 링크와 함께 살펴본다.
8. 결과를 복사하거나 전체 프로젝트를 JSON/Markdown으로 내보낸다.
9. 값은 브라우저 `localStorage`에 프리셋별로 저장하고 JSON으로 다시 불러올 수 있다.

## 3. 기능 요구사항

- 한 화면에 모든 노드를 뿌리지 않고 네 개의 단계 화면으로 분리한다.
- 단계 안의 표준 노드는 포인터/터치로 이동할 수 있다.
- 노드 위치와 편집값을 단계·프리셋별로 자동 저장한다.
- 노드 잠금/해제와 단계별 배치 초기화를 지원한다.
- 프리셋마다 독립된 상태를 유지한다.
- 노드 연결선은 이동 후에도 다시 계산한다.
- 핵심 입력값: 콘셉트, 감정, BPM, 조성, 대표 길이, 금지 요소, 키워드 묶음.
- 파생 결과: 추천 메인 Mureka 프롬프트, 구조 수와 일치하는 트랙 프롬프트, 추천 이미지 프롬프트, 네거티브 프롬프트, 클릭을 부르는 제목 후보, 챕터형 기승전결, 참고 이미지 10개.
- 텍스트 결과는 개별 복사할 수 있다.
- 전체 결과는 JSON과 Markdown으로 내보낼 수 있다.
- 외부 요청, 추적, API 키 입력란이 없는 정적 앱이어야 한다.
- 배포 캘린더와 3단계 검수는 화면 노드에서 제외하고 문서의 고정 체크리스트로 유지한다.
- 참고 이미지는 그대로 재배포하는 자산이 아니라 장면·렌즈·노출·시간대를 연구하는 미리보기이며 원출처로 이동할 수 있어야 한다.

## 4. 권장 길이와 검증 가설

길이는 플랫폼의 공식 규칙이 아니라 표본과 사용 맥락에서 도출한 시작 가설이다.

| 역할 | 길이 | 목적 |
|---|---:|---|
| 파일럿 | 28분 | 제작비를 낮추고 제목·썸네일·첫 30초 반응 확인 |
| 대표편 | 64분 | 약 1시간형 앰비언트/집중 믹스의 익숙한 사용 단위 |
| 확장편 | 128분 | 대표편의 유지율이 확인된 뒤 집중·게임용으로 확장 |

대표편 기본 전개:

- 0:00–3:30: 문턱/콜드 오픈 — 7–12초 안에 영상의 정체성을 보이고, 20초 안에 핵심 음색을 제시한다.
- 3:30–14:00: 공간 확립.
- 14:00–30:00: 탐색과 의식.
- 30:00–42:00: 금지된 사실/추격의 긴장.
- 42:00–54:00: 감정적 절정.
- 54:00–64:00: 잔향과 루프 가능한 귀환.

트랙/챕터는 9–12개, 평균 4–7분, 전환 크로스페이드는 6–12초를 시작값으로 쓴다. 갑작스러운 무음이나 축제형 EDM 드롭은 두 프리셋의 사용 목적과 맞지 않으므로 기본적으로 피한다.

검증 지표는 조회수 단독이 아니라 노출 클릭률, 첫 30초 유지, 평균 시청시간, 핵심 구간 유지율, 재방문 시청자로 둔다. 비슷한 길이의 최근 영상끼리 비교한다.

## 5. 데이터 모델

```text
preset
├── meta: id, name, eyebrow, status
├── theme: subject, setting, ritual, emotion, audiencePromise
├── music: bpm, key, meter, genres, instruments, texture, exclusions
├── visual: subject, setting, camera, palette, texture, exclusions
├── keywords: location, ritual, sensory, emotion, music, discovery
├── duration: pilot, flagship, extended, chapters, crossfade
├── story: six phases with minute ranges and energy values
├── campaign: relative timing, deliverable, goal
├── rights: originality and disclosure rules
└── nodes: id, kind, title, x, y, inputs, outputs
```

저장 스키마:

```text
zerune-studio/v2/<preset-id>
├── version
├── updatedAt
├── fields
├── positions
├── lockedNodes
└── customNodes
```

## 6. 노드 그래프

```text
FLOW 1 · 콘셉트
[콘셉트 브리프] ─> [클릭 제목 방향]

FLOW 2 · 키워드
[장소·의식·감각·정서] ─> [음악 키워드] + [이미지 키워드]

FLOW 3 · 사운드 + 구조
[사운드 설계] ─> [대표편 N개 트랙 구조]

FLOW 4 · 프롬프트
[추천 메인 음악 프롬프트]
          ├─ + [트랙 01 구조] ─> [트랙 프롬프트 01]
          ├─ + [트랙 02 구조] ─> [트랙 프롬프트 02]
          └─ + [트랙 0N 구조] ─> [트랙 프롬프트 0N]
          └─ 동일 장면·정서·색 ─> [추천 이미지 프롬프트] ─> [참고 이미지 10개]
```

화면에는 현재 FLOW의 노드만 보인다. 연결선은 해당 단계 안의 정보 흐름만 간결하게 안내한다.

## 7. 프리셋 A 방향

- 표기: 메인 제목은 검색성을 위해 `평양냉면`, 콘셉트 문구에는 북측 표기감을 살린 `평양랭면`을 병기한다.
- 음악: 68–76 BPM, D minor/D Dorian, dark ambient + minimal dub techno, prepared piano, bowed metal, glass harmonics, 환풍기 저주파, 절제된 오음 음형.
- 이미지: 겨울밤의 삼각형 거대호텔 연회장, 놋그릇 냉면, 성에 낀 들쭉 과일주, 비어 있는 좌석, 유리창 결로, 정치 상징 없음.
- 태도: 실제 북한 음악을 재현한다고 주장하지 않고, 가상의 영화적 공간으로 명확히 다룬다.

## 8. 프리셋 B 방향

- 음악: 92–108 BPM, half-time 감각, F-sharp minor/Phrygian, darksynth + industrial ambient, FM bass, EBM pulse, granular alarms, metallic percussion.
- 이미지: 독창적인 2D 애니메이션 SF 스틸, 익명 사이버네틱 배달원, 빗속 고가철도, 청록/경고색, 대각 구도, 크로매틱 스플릿.
- 금지: 기존 캐릭터, 의상, 로고, 프랜차이즈 멜로디, 공식 스틸처럼 보이는 문구, 특정 스튜디오 화풍 직접 모사.
- 일정: 공식 사이트가 밝힌 `Fall 2026`을 기준으로 상대 일정만 제공한다. 정확한 공개일은 미정으로 표시하고 공식 페이지 확인 노드를 둔다.

## 9. 보안·권리 경계

- 정적 브라우저 앱에 Mureka API 키를 넣지 않는다.
- 생성물의 상업 이용 전에는 사용 시점의 Mureka 약관과 각 생성 도구의 라이선스를 다시 확인한다.
- 프랜차이즈 이름은 리서치/검색 문맥에서만 제한적으로 사용하고, 생성 프롬프트는 독창적이고 일반적인 시각 언어로 작성한다.
- 썸네일과 설명에는 `original / unofficial / not affiliated` 성격을 분명히 한다.
- 실제 정치 인물, 국기, 선전물은 프리셋 기본값에서 제외한다.

## 10. 완료 조건

- 두 프리셋과 네 단계가 전환되고 상태가 서로 섞이지 않는다.
- 현재 단계의 기본 노드가 이동하며 새 위치가 새로고침 후 유지된다.
- 3단계의 트랙 수와 4단계의 트랙 프롬프트 수가 항상 일치한다.
- 메인 음악 프롬프트와 추천 이미지 프롬프트가 시각적으로 강조된다.
- 각 프리셋의 참고 이미지 10개가 미리보기·원출처 링크·참고 관점과 함께 표시된다.
- 편집 → 결과 생성 → 복사 → JSON/Markdown 내보내기가 작동한다.
- 모바일 너비에서 노드 편집과 이동이 가능하다.
- 자동 테스트, 저장소 전체 검사, 실제 브라우저 상호작용 및 시각 검사를 통과한다.
- 사실성, 콘셉트/권리, 기능/UX의 세 검토 결과를 `REVIEW.md`에 남긴다.
